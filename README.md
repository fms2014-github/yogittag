# Front-End Vue Project
## Nodejs설치하기

### Windows
1. [https://nodejs.org/ko/](https://nodejs.org/ko/)에 접속한다.
2. LTS 버전을 다운로드 한다. 
3. 다운로드 받은 파일을 실행하여 설치를 한다.
4. 명령 프롬프트를 실행한 뒤 node를 실행하여 버전확인
## Vue.js설치하기
### CLI를 사용하여 Vue.js 사용하기
Vue CLI를 설치하기 위해선 NPM 모듈이 필요하다. NPM은 Node Package Manager의 약자로 Node.js 내부에 들어있는 패키지 모듈이다. 그렇기 때문에 Node.js 설치 작업이 필요하다.
1. npm 모듈로 vue를 설치한다.
        npm install vue
2. vue/cli를 설치한다.
        npm install -g @vue/cli
## 프로젝트 실행하기
### Windows
1. 명령 프롬프트로 clone한 프로젝트 홈 디렉터리로 이동
2. npm install 입력 후 엔터
3. npm serve run 입력 후 엔터
4. http://localhost:8080/ 주소로 접속
